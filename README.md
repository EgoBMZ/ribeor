## ✦ Overview

Ribeor is more than a portfolio; it's a living sandbox for modern web development. Built with a focus on UI/UX excellence, it features a custom design system, seamless internationalization (i18n), and a fully integrated headless CMS powered by Firebase for real-time project management and analytics.

### Key Features
- **Premium UI/UX:** Custom Aurora backgrounds, glassmorphism, and fluid micro-interactions.
- **Performance First:** Next.js App Router, SSR/SSG, and optimized asset delivery.
- **Admin Dashboard:** Secure authentication (Google) to manage projects, posts, and view live analytics.
- **Internationalization (i18n):** Flawless language switching (EN/ES) without hard reloads.

<br />

## ✦ Architecture & Stack

| Layer | Technology | Description |
| --- | --- | --- |
| **Core** | React 18, Next.js (App Router) | Server Components & optimized rendering. |
| **Language** | TypeScript | Strict type safety across the entire codebase. |
| **Styling** | Tailwind CSS + CSS Modules | Utility-first styling with custom CSS variables. |
| **Backend & Auth** | Firebase (Firestore, Auth) | NoSQL database and secure OAuth integration. |
| **Deployment** | Vercel | Edge network hosting and CI/CD pipelines. |

## ✦ Local Development

To run this project locally, you will need Node.js and a Firebase project.

```bash
# 1. Clone the repository
git clone https://github.com/EgoBMZ/ribeor.git

# 2. Install dependencies
npm install

# 3. Setup environment variables
# Create a .env.local file and add your Firebase and Admin keys

# 4. Start the development server
npm run dev
```
